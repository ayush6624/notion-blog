# Notion Blog

Using Next.js with Notion's API for blogging!
https://blog.ayushgoyal.dev

## Running Locally

1. Installing Dependencies `yarn`
2. Run next in development mode `yarn dev`
3. Build and run in production mode `yarn build && yarn start`
